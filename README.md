# fruits-game

This is a fruits slicing game! Just move the mouse to cut fruits as fast as you can to boost your score! 

## Where to find it?

Just click on [fruits-game](https://yedzinovich.github.io/fruits-game/) and play! 

## Built With

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JQuery</li>
</ul>
